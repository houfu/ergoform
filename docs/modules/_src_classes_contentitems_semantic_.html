parseSemantic

  • parseSemantic(value: string): Semantic
    • Defined in src/classes/ContentItems/Semantic.ts:73

    Utility function to parse a string value to the Semantic type. If there is no match, returns Semantic.none. Note: the value is not required to correspond directly with the Semantic string value and some alternatives are accepted. As the Enum is subject to changes, it is highly recommended for users to refrain from using index values (such as 0, 1 etc) and use more user friendly strings instead.

    Parameters

    • value: string

      String value to parse.

    Returns Semantic

    Matched Semantic from string.
